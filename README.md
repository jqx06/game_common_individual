# 《匹夫》文字冒险游戏

基于 Qt6/C++ 开发的国风异能文字冒险游戏。

---

## 环境配置

- **开发环境**：Qt 6.11 + MinGW 64-bit + C++17
- **构建运行**：用 Qt Creator 打开  game_common_individual.pro

---

## 项目结构

```
/include          存放头文件
/src              存放源文件
/ui               存放ui文件
/resources        
  /asset          存放图片
  /audio          存放音频
```

## 职业

| 职业 | 风格 | 专属技能 |
|------|------|---------|
| **战士** | 凭血肉之躯碾碎一切阻碍 | 威压：震慑 NPC 跳过冲突 |
| **刺客** | 在阴影中裁决生死 | 暗杀：跳过战斗直接消灭目标 |
| **法师** | 咒语既出，凡人仰望天灾 | 洞悉：读取隐藏信息、预知陷阱 |
| **富商** | 钱能通神，买不到就雇人抢 | 买通：花费金币策反敌方 |
| **盗贼** | 没有锁打不开，没有路逃不掉 | 潜行：绕过战斗获取战利品 |

不同职业在 NPC 对话中拥有不同的文本、选项和剧情分支。

## 场景

| 场景 | 简介 |
|------|------|
| **桃源村** | 南境边陲小村，故事起点。NPC：覃伯、庄叔、田姨、神秘旅人 |
| **幽暗密林** | 巨木遮天，山贼出没。NPC：猎户老赵、山贼探子、受伤商人 |
| **黑风洞** | 山贼大本营，千年妖骸深埋。NPC：疤面、瑶儿、隐世老者 |
| **青石镇** | 南境最大镇落，暗流涌动。NPC：沈大人、老鬼、铁峰、血狼 |
| **玄铁城** | 南境心脏，铁壁高墙。NPC：萧正阳、卿伊瑟、赤眉、青玉案 |

## 双主线

- **侠道**：助村民剿山贼 → 救人质 → 获封赏 → 揭露阴谋 → 保境安民
- **枭雄**：联山贼 → 纳投名状 → 渗透镇落 → 夺权 → 取而代之
